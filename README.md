# INF-512 - Asignación Unidad 2
## Encapsulación y Constructores en C#

### Descripción
Implementación de ejercicios de programación orientada a objetos:
- **Ejercicio 1:** Clase Producto con encapsulación y validaciones
- **Ejercicio 2:** Clase Circulo con constructores

### Archivos del Proyecto
- `Program.cs` - Código fuente principal
- `Diagrama_UML_Circulo.png` - Diagrama UML de la clase Circulo
- `ejercicio_AP_2.1.csproj` - Archivo de proyecto
- `ejercicio_AP_2.1.sln` - Solución de Visual Studio

### Cómo Ejecutar
```bash
dotnet run
```

### Características Implementadas
✅ Encapsulación con atributos privados  
✅ Getters y setters con validaciones  
✅ Constructores (sin argumentos y parametrizados)  
✅ Validación de datos en setters y constructores  
✅ Método DescontarStock con validaciones  
✅ Cálculo de área y circunferencia del círculo  

### Validaciones
- Precio no puede ser negativo
- Stock no puede ser negativo
- Radio debe ser mayor que 0
- No se puede descontar más stock del disponible
